# 🌤 Async Weather Tracker

A simple web application built using **HTML, CSS, and Vanilla JavaScript** to demonstrate asynchronous programming concepts in JavaScript.

This project was created as part of a Web Development lab assignment to understand how JavaScript behaves at runtime when working with APIs and asynchronous code.

## 🚀 Features

- 🔎 Search weather by city name
- 🌡 Display temperature in Celsius
- 🌥 Show weather description, humidity, and wind speed
- 💾 Save searched cities using Local Storage
- 🔁 Click previous searches to re-fetch weather data
- 📜 Demonstrate Event Loop execution order using console logs
- ⚠ Proper error handling for invalid input and API errors

## 🛠 Technologies Used

- HTML5  
- CSS3  
- Vanilla JavaScript (ES6)  
- OpenWeatherMap API  

No frameworks or external libraries were used.
